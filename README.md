# orchedule
오케스트라를 위한 연습 일정 알림, 출석체크 및 자료 관리 사이트

## 🖥️ 프로젝트 소개
제가 활동했던 오케스트라 단체에서 사용하고 싶어서 만든 연습 일정 알림 및 자료 관리 서비스입니다. 
원래 네이버 밴드를 통해 관리하고 있었고 연습 일정과 출석 확인, 필요한 자료 찾기 기능 등 불편했던 기능들 위주로 개발을 시작하게 되었습니다.
